# Portfolio - Muhammad Ganesar Zena

## Cara Menjalankan (How to Run)

### Syarat / Prerequisites
- Node.js versi 18 atau lebih baru: https://nodejs.org
- npm (sudah termasuk di Node.js)

### Langkah / Steps

1. Buka terminal / command prompt di folder ini
2. Install dependencies:
   ```
   npm install
   ```
3. Jalankan dev server:
   ```
   npm run dev
   ```
4. Buka browser dan akses: **http://localhost:5173**

### Build untuk Production
```
npm run build
npm run preview
```
File hasil build ada di folder `dist/`.

Built with React + Vite + Tailwind CSS + Framer Motion
